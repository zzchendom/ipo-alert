# 股票申购提醒（A 股）

每天开机后，如果今日有 A 股新股可申购，屏幕右下角会弹出一个**半透明浮窗**；点击它打开 HTML 报告，关闭后桌面**完全恢复干净**，直到下一天或下一次有申购。

## 文件
| 文件 | 作用 |
|---|---|
| `main.py` | 入口 + 浮窗 + 状态管理 |
| `fetcher.py` | 拉东方财富新股申购数据 + 收益估算 |
| `report.py` | 生成 HTML 报告 |
| `install.vbs` | 注册开机启动（双击运行） |
| `uninstall.vbs` | 移除开机启动 |
| `test.bat` | 用 mock 数据测试浮窗（不影响"今日已读"） |
| `run_today.bat` | 立即拉真实数据并显示浮窗 |
| `state.json` | 记录"今日已读"，自动生成 |
| `report.html` | 生成的报告，点击浮窗时刷新 |

## 使用
1. 双击 `test.bat` —— 应该看到右下角弹出红色浮窗"今日 3 只新股可申购 · 🔥 1 只大肉签"。点击它会在浏览器打开报告。报告关掉后浮窗也消失。
2. 满意后双击 `install.vbs` —— 注册开机启动。
3. 之后每次开机：
   - 今天没新股 → 静默退出，桌面无任何变化
   - 已经看过/标"不再提醒" → 静默退出
   - 有新股且未读 → 浮窗出现

## 浮窗交互
- **点击** → 打开报告 + 浮窗消失 + 标记今日已读
- **拖动** → 调整位置
- **右键** → "查看报告" / "今日不再提醒" / "退出（不标已读，下次再弹）"

## 颜色含义
- 🔴 红色：含中性预期 ≥ 3 万的"重点关注"新股（大肉签）
- 🔵 蓝色：含中性预期 ≥ 1 万的"建议必申"新股
- ⚫ 灰色：普通新股

## 收益估算口径
单签盈利 = 单签金额 × 预估首日涨幅，分板块和赛道：
- 沪/深主板：保守 50% / 中性 100% / 乐观 180%
- 创业板/科创板：保守 60% / 中性 140% / 乐观 250%
- 北交所：保守 50% / 中性 120% / 乐观 250%
- 半导体 / AI / 算力 / 创新药 / 机器人等热门赛道：每档 +20% / +50% / +100%
- 发行 PE 显著低于行业 PE（< 70%）：每档再 +10% / +30% / +50%

数据仅供参考，自负盈亏。

## 实际中签率回顾
浮窗每次启动时，会检查历史申购但还没"看过中签率"的股票：
- T 日打新 → T+2 出中签率 → T+2 之后第一次开机的浮窗里会带"📊 回顾 N 只中签率"
- 看过一次（点击或"今日不再提醒"）就标记 reviewed，**不再重复显示**
- 报告顶部的紫色卡片对比"预测 vs 实际"中签率，给出偏差注释（"高于预测 / 低于预测 / 接近预测"）
- 超过 30 天还没出中签率的会被清理（避免无限增长）

## 北交所开关
默认**显示**北交所新股。注意北交所开户门槛较高（日均资产 ≥ 50 万 + 2 年交易经验），
若你没开通北交所权限、不想看到这类新股，编辑 `fetcher.py` 顶部改为：
```python
INCLUDE_BJ = False
```

## 托盘常驻版（推荐，解决"忘了关机第二天没提醒"）

旧版 `main.py` 只在**开机那一刻**跑一次：不重启电脑就不会再检查。`tray.py` 改为**常驻系统托盘**：

- **开机/启动时立即检查一次，之后每天早上 9:00 自动复查一次**（定时器在睡眠时暂停、唤醒后立即补触发）→ 不依赖重启，根治"忘关机"。
- 注：新托盘图标常被 Windows 10 藏进 **^** 溢出区，点开 **^** 把"申"拖到外面即可常显。
- 右下角时钟旁常驻一个**"申"字小图标**：**左键点它 = 随时查看今日报告**（你的"手动按钮"）。
- 桌面全程干净，投屏时只有一个不起眼的托盘图标。
- 有内容时图标变色 + 加金色角标；**自动提示每天只触发一次**。
- **每天第一次点开报告后，"申"图标立即消失**（投屏时桌面/托盘都干净）；进程在后台静默常驻，**第二天自动把图标放回来**——不依赖你重启电脑。浏览器里的报告不受影响，可继续看。
- 想关掉的话右键"退出"即可；若只是不想被弹窗打扰但保留图标，选"今天不再主动弹窗"。

### 提示方式（右键菜单"提示方式"可切换，记在 `state.json` 的 `notify_mode`）
| 模式 | 行为 | 适合 |
|---|---|---|
| `icon`（默认） | 仅图标变色 + 角标，不弹窗 | 经常投屏、要最干净 |
| `float` | 有新股弹一次半透明浮窗（沿用旧浮窗） | 想要明显提醒 |
| `balloon` | 右下角系统气泡，几秒自动消失 | 介于两者之间 |

### 托盘版文件
| 文件 | 作用 |
|---|---|
| `tray.py` | 托盘常驻入口（复用 `fetcher`/`report`/旧 `Floater`） |
| `tray_test.bat` | mock 数据测试托盘（`--mock --force`，带控制台，关窗口即结束） |
| `install_tray.vbs` | 把开机启动换成托盘常驻版（覆盖旧 `main.py` 启动项） |

### 使用
1. 双击 `tray_test.bat` 体验：右下角出现"申"图标，右键里切换三种提示方式、点"立即检查"看效果。
2. 满意后双击 `install_tray.vbs` 注册开机启动（自动覆盖旧的 `main.py` 启动项，不会重复启动）。
3. 卸载同样用 `uninstall.vbs`。

## 卸载
双击 `uninstall.vbs` 移除 .vbs 启动项；如果是用 PowerShell .lnk 方式注册的，运行：
```
Remove-Item "$env:APPDATA\Microsoft\Windows\Start Menu\Programs\Startup\IPO_Alert.lnk"
```
文件夹整体删除即彻底卸载。
