# 股票申购提醒（A 股）

每天开机后，如果今日有 A 股新股可申购，屏幕右下角会弹出一个**半透明浮窗**；点击它打开 HTML 报告，关闭后桌面**完全恢复干净**，直到下一天或下一次有申购。

## 文件
| 文件 | 作用 |
|---|---|
| `main.py` | 入口 + 浮窗 + 状态管理 |
| `fetcher.py` | 拉东方财富新股申购数据 + 收益估算 |
| `report.py` | 生成 HTML 报告 |
| `install.vbs` | 注册开机启动（双击运行） |
| `uninstall.vbs` | 移除开机启动 |
| `test.bat` | 用 mock 数据测试浮窗（不影响"今日已读"） |
| `run_today.bat` | 立即拉真实数据并显示浮窗 |
| `state.json` | 记录"今日已读"，自动生成 |
| `report.html` | 生成的报告，点击浮窗时刷新 |

## 使用
1. 双击 `test.bat` —— 应该看到右下角弹出红色浮窗"今日 3 只新股可申购 · 🔥 1 只大肉签"。点击它会在浏览器打开报告。报告关掉后浮窗也消失。
2. 满意后双击 `install.vbs` —— 注册开机启动。
3. 之后每次开机：
   - 今天没新股 → 静默退出，桌面无任何变化
   - 已经看过/标"不再提醒" → 静默退出
   - 有新股且未读 → 浮窗出现

## 浮窗交互
- **点击** → 打开报告 + 浮窗消失 + 标记今日已读
- **拖动** → 调整位置
- **右键** → "查看报告" / "今日不再提醒" / "退出（不标已读，下次再弹）"

## 颜色含义
- 🔴 红色：含中性预期 ≥ 3 万的"重点关注"新股（大肉签）
- 🔵 蓝色：含中性预期 ≥ 1 万的"建议必申"新股
- ⚫ 灰色：普通新股

## 收益估算口径
单签盈利 = 单签金额 × 预估首日涨幅，分板块和赛道：
- 沪/深主板：保守 50% / 中性 100% / 乐观 180%
- 创业板/科创板：保守 60% / 中性 140% / 乐观 250%
- 北交所：保守 50% / 中性 120% / 乐观 250%
- 半导体 / AI / 算力 / 创新药 / 机器人等热门赛道：每档 +20% / +50% / +100%
- 发行 PE 显著低于行业 PE（< 70%）：每档再 +10% / +30% / +50%

数据仅供参考，自负盈亏。

## 实际中签率回顾
浮窗每次启动时，会检查历史申购但还没"看过中签率"的股票：
- T 日打新 → T+2 出中签率 → T+2 之后第一次开机的浮窗里会带"📊 回顾 N 只中签率"
- 看过一次（点击或"今日不再提醒"）就标记 reviewed，**不再重复显示**
- 报告顶部的紫色卡片对比"预测 vs 实际"中签率，给出偏差注释（"高于预测 / 低于预测 / 接近预测"）
- 超过 30 天还没出中签率的会被清理（避免无限增长）

## 北交所开关
默认**不显示**北交所新股（北交所开户门槛：日均资产 ≥ 50 万 + 2 年交易经验）。
若已开通北交所权限，编辑 `fetcher.py` 顶部：
```python
INCLUDE_BJ = True
```

## 卸载
双击 `uninstall.vbs` 移除 .vbs 启动项；如果是用 PowerShell .lnk 方式注册的，运行：
```
Remove-Item "$env:APPDATA\Microsoft\Windows\Start Menu\Programs\Startup\IPO_Alert.lnk"
```
文件夹整体删除即彻底卸载。
